## 说明
    1). 此项目是关于使用webpack进行模块化打包的教学案例
    2). 一共有三个测试单元: 由浅入深, 层层递进
    
## demo1
    1). 理解项目的模块化打包
    2). 学会webpack的基本使用
    3). 理解webpack的4个核心概念
    
## demo2
    1). 利用webpack打包项目中的各种资源
        * JS(ES6)
        * CSS
        * 图片
        * JSON
    2). 加深对loader和plugin的理解
    
## demo3
    1). 2种环境不同的配置
    2). 开发环境: 实现live-reload, 开启sourceMap调试
    3). 生产环境: 单独打包css, 打包文件缓存处理
